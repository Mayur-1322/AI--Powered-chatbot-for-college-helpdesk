# 🎓 AI-Powered Student Help Chatbot for College Helpdesk

An AI-powered chatbot system developed to automate student support by answering college-related queries such as admissions, fees, hostel information, courses, and FAQs. This project reduces manual effort and provides instant support through an interactive chatbot interface.

## 🚀 Live Demo

Deployed Link:  
https://student-help-chatbot.onrender.com/

---

# 📌 Project Overview

Student Help Chatbot is an AI-based helpdesk solution that assists students by providing instant responses to common college queries.

The system uses:

- Natural Language Processing (NLP)
- Web Scraping
- AI Models
- JSON-based Knowledge Storage
- Web Interface for User Interaction

The chatbot works continuously and helps institutions improve student support efficiency.

# ⚙ How System Works

1. User enters a query into chatbot  
2. NLP processes query  
3. System searches generated data  
4. Best response is returned  

Data generation flow:

1. User provides college website URL  
2. Web scraping extracts information  
3. Data cleaned and stored in JSON  
4. Chatbot uses generated dataset

# ▶ Installation

Clone Repository:

```bash
git clone YOUR_REPOSITORY_LINK
```

Move into project:

```bash
cd helpdesk
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000/
```
